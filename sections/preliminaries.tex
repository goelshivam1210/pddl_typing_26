\section{Preliminaries}
\label{sec:preliminaries}

\subsection{Large Language Model Queries}
\label{subsec:llm_query} Given a prompt $\rho$, set of tuples
containing matched pairs of objects $o \in O$ and types $t \in T$.

\subsection{Problem Formulation}
\label{subsec:problem_formulation} Given an unsolvable planning domain
$D$ and a planning problem $P = \langle O, I, G \rangle$, our goal is
to identify untyped objects $O' \subseteq O$ that, assigned a type $t
\in T$ make the problem solving in the context of the goal $G$.
